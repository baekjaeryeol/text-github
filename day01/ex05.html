table

table 및 관련 태그와 속성
  • table : 표를 작성할 때 사용하는 태그
    • width : 가로 너비
    • height : 세로 높이
    • border : 테두리
    • bgcolor : 배경색
    • align : 테이블 자체 정렬 (문서 내에서 테이블의 위치)
    • valign : 수직 정렬 (top, middle, bottom)
    • cellpadding : 한 칸과 내용과의 간격
    • cellspacing : 각각 칸 끼리의 간격

  • tr : table row cell, 한 줄을 표현하는 태그
    • width : 행의 가로 너비
    • height : 행의 세로 높이
    • bgcolor : 배경색
    • align : 행의 수평 정렬 (내부 내용)
    • valign : 행의 수직 정렬

  • td : table data cell, 한 칸을 표현하는 태그, 줄 안에 배치되어야 한다
    • width : 셀의 가로 너비
    • height : 셀의 세로 높이
    • bgcolor : 배경색
    • align : 셀의 수평 정렬 (내부 내용)
    • valign : 셀의 수직 정렬
    • colspan : 지정된 열의 수 만큼 합치기 (가로 확장)
    • rowspan : 지정된 행의 수 만큼 합치기 (세로 확장)

  • th : table header cell, td와 비슷하지만 굵게 처리되며 가운데 정렬된다
