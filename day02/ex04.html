기초적인 style 속성

속성 예시 설명
display display: block 요소의 화면 표시 형태를 변경할 수 있다
block : 한줄 자리차지, 크기 변경 가능
inline : 내용만큼만 자리차지, 크기 변경 불가
inline-block : 크기 지정 가능, 한 줄 안에 여러 요소 배치 가능
flex : 내부 요소의 정렬을 위한 display 속성. 추가 style을 적용해야 한다
none : 요소가 문서에서 시각적으로 사라지며, 자리차지도 없다
width width: 100px 요소의 가로 너비를 지정한다. px, em, rem, % 등으로 표현가능
height height: 100px 요소의 세로 높이를 지정한다. px, em, rem, % 등으로 표현가능
color color: black; color: #00ffff; 기본색(글자색)을 지정한다. 색상이름이나 rgb코드를 사용한다
#000, #123456, rgb(0, 255, 255)
red, green, blue의 수치를 순서대로 0 ~ 255 사이의 값을 지정한다
#으로 시작할 때는 rgb수치를 16진수로 작성 (0 ~ f)
background-color background-color: salmon 배경색을 지정한다. 색상이름이나 rgb코드를 사용한다
border border: 1px solid black 테두리의 두께, 선 형태, 색상을 순서대로 지정한다
두께는 px 단위로 지정하면 된다
형태는 solid, dotted, double, dashed, none
색상은 color 지정 방식과 동일하다
margin margin: 10px; margin: 20px auto; margin: 5px 10px 5px 5px 테두리 바깥 여백을 지정한다
4개를 순서대로 작성하면 top -> right -> bottom -> left 순서이다
2개를 순서대로 작성하면 상하 -> 좌우 순서이다
1개만 작성하면 상하좌우 모두 같은 여백을 적용한다
padding margin과 사용법은 동일함 테두리 안쪽 여백을 지정한다. 테두리와 내용 사이의 여백
