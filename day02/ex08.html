1. 태그 선택자

태그이름 { 속성: 값; }
ol { list-style: none; }
문서 내의 지정한 태그가 모두 적용된다

2. 클래스 선택자

.클래스이름 { 속성: 값; }
.new { color: yellow; }
문서 내에서 태그에 상관없이 지정한 클래스에 적용된다

3. ID 선택자

#ID이름 { 속성: 값; }
#logo { font-size: 40px; }
문서 내에서 태그에 상관없이 지정한 ID에 적용된다
태그에 id를 부여할 때는 문서 전체에서 유일한 이름으로 부여하도록 한다
id는 다른 태그와 중복되면 안된다

4. 자손 선택자

부모요소 > 자식요소 { 속성: 값; }
ol > li { color: white; }
문서 내에서 특정 부모요소 바로 밑에 속해있는 자식 요소에 적용된다
같은 태그이거나, 같은 클래스이더라도, 지정한 부모요소가 일치할 때에만 적용 가능

5. 후손 선택자

부모요소 후손요소 { 속성: 값; }
nav li { border-bottom: 1px solid black; }
직계 하위 요소는 아니지만, 단계가 멀리 있더라도, 내부에 속해있으면 후손 요소에 속성을 적용할 수 있다

6. 조건 선택자

요소:조건 { 속성: 값; }
a:hover { text-decoration: underline; }
특정 요소에 특정 조건을 만족하는 경우에 적용하는 속성이다
:active - 마우스 클릭이 눌러져있는 경우에 적용되는 속성
:hover - 마우스 포인터가 요소 위에 올라가 있을 때
:focus - 클릭하거나 키보드 탭키에 의해서 요소가 선택되어 있는 경우
:visited - 방문했던 사이트의 주소인 경우
